## Getting Started

### Requirements
- node >= `12.18.0`
- yarn >= `1.22.5`

### Installation
```bash
# Clone this repository
$ git clone https://github.com/tshamz/kingmaker

# Install dependencies (must use yarn)
$ yarn install
```

### Environment Variables
```bash
# Create .env file
$ touch .env
```
Add values for the following env vars:
  - `FIRST`
  - `SECOND`
  - `THIRD`


### Usage
To start developing
```bash
$ yarn dev
```
